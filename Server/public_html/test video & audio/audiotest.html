Streaming Audio with Rackspace CDN

Instructions:

  1. After you have uploaded the audio files to your Cloud Files account, view the links for the CDN-enable content.
  2. Use the HTTP link from the .mp3 file and add it to the code.
  3. Use the HTTP link from the .ogg file and add it to the code.
  4. Upload the web page with the links to the audio files on your account.

Sample code: 

<audio controls >

<source src="full/link/to/http/mp3/audio/file.mp3" type="audio/mpeg" />

<source src="full/link/to/http/ogg/audio/file.ogg" type="audio/ogg" />

<em>Sorry, your browser doesn't support HTML5 audio.</em>

</audio>
